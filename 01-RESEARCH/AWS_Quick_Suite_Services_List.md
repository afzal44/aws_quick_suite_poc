## Core Services & Components

### 1. **Quick Index** (Knowledge Foundation)
**Purpose:** Unified knowledge repository that indexes and connects all enterprise data

**Capabilities:**
- ✅ Automatic indexing of uploaded files and unstructured data
- ✅ Connects to 40+ application integrations
- ✅ Unified search across all data sources
- ✅ Real-time data synchronization
- ✅ Metadata extraction and cataloging
- ✅ Data lineage tracking
- ✅ Version control for documents

**Supported Data Sources:**
- **Cloud Storage:** Amazon S3, Google Drive, Microsoft SharePoint, Dropbox, Box
- **Databases:** Amazon Redshift, Snowflake, PostgreSQL, MySQL, SQL Server
- **Data Warehouses:** Amazon Redshift, Snowflake, BigQuery, Azure Synapse
- **Query Engines:** Amazon Athena, Presto, Apache Spark
- **Documents:** PDF, Word, Excel, PowerPoint, Text files
- **Email:** Microsoft Exchange, Gmail, Outlook
- **Collaboration:** Slack, Microsoft Teams
- **Project Management:** Jira, Asana, Monday.com
- **CRM:** Salesforce, HubSpot, Dynamics 365
- **Support:** Zendesk, ServiceNow, Freshdesk

**Pricing Model:**
- Storage-based: $0.50/GB/month for indexed data
- Includes automatic refresh and synchronization

- ✅ Version control
- ✅ Testing and debugging tools

**Workflow Components:**
- **Input Steps:** Gather information from users or systems
- **Reasoning Groups:** AI-powered processing and decision-making
- **Output Steps:** Generate results, send notifications, update systems
- **Approval Steps:** Human-in-the-loop for critical decisions
- **Error Handling:** Retry logic, fallback actions, notifications

**Use Cases:**
- Automated reporting
- Data validation and cleansing
- Notification and alert systems
- Simple approval workflows
- Data synchronization between systems
- Content generation
- File processing and organization

**Pricing Model:**
- Included in Quick Suite subscription

### 7. **Chat Agents** (Specialized AI Assistants)
**Purpose:** Conversational AI agents configured for specific business functions

**Types:**

#### **Built-in Insights Agent**
- Answers questions across all connected data
- General-purpose assistant
- No configuration required

#### **Custom Chat Agents**
- Configured with specific expertise
- Connected to curated data sources
- Tailored instructions and behavior
- Department or function-specific

**Configuration Options:**
- **Knowledge Sources:** Spaces, tables, documents, APIs
- **Capabilities:** Query data, generate reports, execute actions
- **Permissions:** Read/write access to specific resources
- **Personality:** Tone, style, response format
- **Actions:** Create tickets, send alerts, update records
- **Escalation:** When to involve humans

**Example Agents:**
- Sales Assistant (product catalog, pricing, CRM)
- Compliance Agent (regulations, policies, approvals)
- HR Assistant (policies, benefits, onboarding)
- IT Support Agent (troubleshooting, documentation)
- Finance Agent (accounting policies, expense rules)
- Customer Service Agent (product info, order status)

**Pricing Model:**
- Included in Quick Suite subscription
- Usage-based for high-volume scenarios

## Pricing Summary

| Service | Pricing Model | Typical Cost |
|---------|--------------|--------------|
| **Quick Suite Enterprise** | Per user/month | $35/user (business), $75/user (power) |
| **Quick Index** | Storage-based | $0.50/GB/month |
| **Quick Research** | Per report | $5/report (standard) |
| **Quick Flows** | Execution-based | $0.10/run |
| **Quick Automate** | Execution-based | $0.10-0.50/run |
| **Premium Data Sources** | Subscription | Variable (per source) |
| **AWS Infrastructure** | Standard AWS pricing | Redshift, S3, Lambda, etc. |

**Example Annual Cost (50 users, 5TB data):**
- Licensing: $21,000 (business users) + $4,500 (power users)
- Quick Index: $30,000 (5TB storage)
- Quick Research: $30,000 (500 reports/month)
- Quick Automate: $12,000 (10,000 runs/month)
- **Total: ~$97,500/year** (plus AWS infrastructure)
